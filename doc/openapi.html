cert

Operations related to X.509 certificates

Lint a Certificate

Lints a X.509 certificate

Request Body schema: application/x-www-form-urlencoded
required

The parameters for the linting request

b64input
required
string

The Base64 or PEM-encoded input

format
string (ResponseFormat)
Default: "json"
Enum: "json" "html" "text"

The response format

profile
string
Default: "autodetect"

The name of a profile

severity
string (FindingSeverity)
Enum: "meta" "debug" "info" "notice" "warning" "error" "bug" "fatal"

The severity of a linter finding

Responses
